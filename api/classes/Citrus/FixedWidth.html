Class Citrus::FixedWidth

  1. lib/citrus.rb
Parent: Object

A FixedWidth is a Terminal that matches based on its length. The Citrus notation is any sequence of characters enclosed in either single or double quotes, e.g.: 

'expr'
"expr"
